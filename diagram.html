⚡ How data flows from the internet → your prediction
1
📡 Binance WebSocket connects on page load
wss://stream.binance.com → proxied via Vite → binanceWS.js
Fires every trade. Updates price state in real-time. Fallback to REST polling if WS drops.
2
📊 Fetch Data button → 6 timeframes in parallel
binanceApi.js calls /klines?symbol=BTCUSDT&interval=1m/5m/15m/30m/1h/4h
Returns 200 candles each. Runs indicators.js on every timeframe: MACD · RSI · StochRSI · Bollinger Bands · Supertrend · VWAP. Stored in tfData state.
3
🎯 ⚡ Auto button → 2-phase Polymarket fetch
Phase 1 (Gamma API): finds active BTC UP/DOWN market, scores candidates by time-to-expiry + liquidity + has $ threshold. Extracts token IDs.
Phase 2 (CLOB API): uses token IDs to get real live upOdds / downOdds from actual traded price. Then non-blocking: fetches spread + 1h price history.
Stores: upOdds dnOdds threshold spreadData oddsHistory
4
🐋 Whale Monitor → fetches every 60 seconds
whaleMonitor.js fetches large BTC transactions. Classifies each as BULLISH (exchange withdrawal) or BEARISH (exchange deposit). Computes whaleSentiment.score and whaleSentiment.label.
5
⚡ signalEngine.js → builds the main prediction
Runs every time any input changes (price, odds, weights, whale). Scores 5 factors:
Market Odds ≥53¢ Price Buffer 5m MACD 1m MACD Multi-TF Vote
Need 3+/5 to get UP or DOWN. Then adjusts confidence with whale, StochRSI, spread, BB, Supertrend. Hard-blocks: Dangerous · Whale Dom · Low Liq · Coin flip · StochRSI extreme.
6
🤖 predictionModel.js → AI momentum cross-check
Parallel to signalEngine. Weighs 4H (3pts) · 1H+VWAP (3pts) · 15m (1.5pts) · 5m · 1m momentum · RSI · BB · StochRSI.
Returns dir: UP/DOWN/SIDEWAYS + confidence. Shown as "AI Momentum" widget.
When a bet resolves → updateWeights() reinforces or penalises each feature.
7
💾 Result rendered + persisted to localStorage
Final pred and aiPred shown in Dashboard. Bet placed → balance deducted → stored.
On every change: weights balance betHistory paperBets aiLog all write to localStorage → survives page refresh and app restart.
🤖 What happens every 5-minute Polymarket round
0
T = 0:00 — New round starts
Polymarket publishes a new market: "Will BTC be above $69,500 at 8:05 PM?"
Bot resets fetchedThisRound = false, bettedThisRound = false.
Status: WATCHING — timer counts down from 5:00.
1
T = 4:00–4:05 (55–65s before end)
Bot fires fetchData() — refreshes all 6 timeframes from Binance.
Sets fetchedThisRound = true so it doesn't fetch again this round.
Status: ANALYZING
2
T = 4:10–4:20 (40–50s before end)
Decision moment. Bot reads current pred signal.

Skip if any of: signal is NO BET · balance < bet size · bettedThisRound already true
Bet if: signal is UP or DOWN with score ≥3/5

Sets bettedThisRound = true. Deducts bet from balance. Stores as pending bet.
Status: BETTING or SKIP logged
3
T = 4:50–5:00 (1–5s before end)
Auto-resolution. Bot reads live price from WebSocket.

If threshold set (real Polymarket): UP wins if price > threshold, DOWN wins if price < threshold.
If no threshold (Smart Fill): compares exit price vs entry price.

Status: WIN ✅ or LOSS ❌ logged to Bot Activity Log.
4
Immediately after — AI learns
updateWeights(weights, features, won, conf) is called.
Each signal feature that fired gets its weight bumped (win) or reduced (loss).
Higher confidence bets have more impact on weight changes (adaptive learning rate).
Wrong bets cost 20% more than correct bets earn — discourages overconfidence.
Balance, betHistory, weights all written to localStorage. Round complete. 🐾
⚡ Signal Scoring — How UP / DOWN / NO BET is decided
5 SCORED FACTORS
1
Market Odds ≥ 53¢
Real edge over 50/50. From Polymarket.
2
Price Buffer (side of threshold)
Price above threshold for UP, below for DOWN.
3
5m MACD direction
Must agree with bet direction.
4
1m MACD direction
Short-term momentum agrees.
5
Multi-TF Weighted Vote
4H×2.5 + 1H×1.5 + 30m/15m/5m/1m weighted majority.
CONFIDENCE ADJUSTMENTS
Base conf by score
3/5 → 55% · 4/5 → 62% · 5/5 → 67%
+Boosts
TF vote strength · Whale aligned · Tight spread · BB extreme · Supertrend agrees
−Penalties
StochRSI ceiling (−10) · Floor + bullish 4H (−12) · Wide spread · Crowd against bet
🚫 Hard blocks (always NO BET)
Dangerous flag · Whale Dom · Low Liq · Coin flip 48–52¢ · StochRSI ≤1 or ≥99 · conf <50%
